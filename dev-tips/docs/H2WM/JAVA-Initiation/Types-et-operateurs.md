# Types et Opérateurs

## Déclaration

```JAVA
package module3_parite;

public class MainParite {
	
	public static void main(String[] args) {
	
	int numSaisi = 12;
    }
}
```

## Les différents types : 
- int : entier
- float : 6 chiffres après la virgule
- float z = 12,5f (pour valider le type float)
- double : 15 ch après la virgule
- boolean : True or False
- char : caractère
- String : avec un S majusculre : texte

## Les opérateurs de comparaison:
* == : égal
* != : différent
* ">", ">=" : supérieur, supérieur égal
* .equals : pour String
* && : et
* || : ou
* "+" : pour concaténer
